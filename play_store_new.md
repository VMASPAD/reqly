# Reqly: Professional API Testing Tool

Short description (max 80 chars)
- Complete API testing platform with collections, environments & scripts.

Full description
Reqly is a comprehensive, professional-grade API testing platform that brings enterprise-level functionality to developers and QA teams. Build, organize, test, and automate your API workflows with powerful tools designed for modern development teams.

🚀 Core Features:
- Send HTTP requests: GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS
- Beautiful JSON editor with syntax highlighting and formatting
- Advanced request builder with headers, params, and authentication
- Smart response viewer with Pretty/Raw modes for JSON, HTML, XML, images
- Real-time status, response time, and size metrics
- Tabbed interface for managing multiple requests simultaneously

📁 Organization & Management:
- Collections: Organize requests into logical groups and folders
- Saved Requests: Build a library of reusable API calls
- Environment Variables: Manage different configurations (dev, staging, prod)
- Variable Substitution: Dynamic values with {{variable}} syntax
- Import/Export: Share collections and environments with your team

🧪 Advanced Testing:
- Test Scripts: Write JavaScript tests using Postman-compatible pm API
- Pre-request Scripts: Execute code before sending requests
- Test Results: Detailed pass/fail reports with timing and assertions
- Environment Integration: Access and modify variables during test execution
- Automated Validation: Assert response status, headers, and body content

🔧 Developer Experience:
- Proxy Support: Optional secure forwarding with custom headers
- Request History: Track and replay previous API calls
- Dark Mode: Easy on the eyes for long coding sessions
- Keyboard Shortcuts: Power-user friendly (⌘/Ctrl+Enter to Send)
- Mobile-Optimized: Clean, responsive interface that works everywhere

💼 Professional Workflow:
- Team Collaboration: Share collections and environments
- CI/CD Ready: Export tests for automated testing pipelines
- Documentation: Auto-generate request documentation
- Version Control: Track changes to your API test suites

- No sign‑in required.
- No personal information is collected by the app.
- Only the endpoints you choose are contacted.
- All data stays on your device - collections, environments, and test results are stored locally.

- No tracking, no analytics, no data mining - just pure functionality.
